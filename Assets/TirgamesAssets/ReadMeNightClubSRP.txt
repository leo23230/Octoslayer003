Night Club v1.2 (SRP version)

HDRP Setup:
- Import NightClubHDRP unitypackage from NightClubSRPPackages folder to new HDRP template project
- In Project Settings->Graphics select NightClubHDRenderPipeline in SRP settings

URP Setup:
- Import NightClubURP unitypackage from NightClubSRPPackages folder to new URP template project
- Please note since URP doesn't support light cookies, Mirror ball's effect won't work in URP version

Night Club main scenes:
- TirgamesAssets/NightClubs/NightClub/Scenes/NightClubScene: Complete Night Club game level. 
- TirgamesAssets/NightClubs/NightClub/Scenes/NightClubRooms: Set of various rooms such as Dance, Strip, Bar, Restrooms
